# Healthcare Metrics Calculator

A Python-based tool for calculating essential healthcare metrics such as BMI, Cardiac Output, and Body Surface Area (BSA).

## Overview
This project focuses on applying basic programming concepts to real-world healthcare calculations. It allows users to input personal health data and compute clinically relevant values.

## Features
- BMI calculation with category classification  
- Cardiac Output calculation (Heart Rate × Stroke Volume)  
- Body Surface Area (BSA) using Du Bois formula  

## Example
Input:
- Height: 1.52 m  
- Weight: 47.7 kg  
- Heart Rate: 72 bpm  
- Stroke Volume: 70 ml  

Output:
- BMI: 20.64 (Normal)  
- Cardiac Output: 5.04 L/min  
- BSA: 1.45 m²  

## Technologies Used
- Python  

## Purpose
The goal of this project is to demonstrate how healthcare-related calculations can be implemented using basic programming and structured logic.
